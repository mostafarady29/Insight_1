**Insight** is a full-stack scientific research management platform designed to provide researchers, academics, and enthusiasts with a professional and highly structured environment to explore, analyze, and interact with scientific research content. The platform combines  **modern front-end design** ,  **robust backend services** , and **AI-powered assistance** to deliver a seamless research experience.

## Overview

Insight enables users to:

* Discover and search scientific papers across multiple fields.
* Explore authors, research fields, and detailed paper content.
* Access personalized paper recommendations based on user activity.
* Interact with an AI assistant for research queries and PDF-based question answering.
* Track downloads, reviews, and user engagement statistics.

The platform emphasizes a  **clean, professional, and scientific UI** , combined with a secure and scalable backend that manages authentication, data integrity, and AI integrations.

## Key Features

* **User Authentication:** Secure registration, login, and JWT-based protected routes.
* **Research Paper Management:** Pagination, field-based filtering, paper statistics, reviews, and downloads logging.
* **Authors & Fields:** Detailed profiles and analytics, including top-rated papers and field statistics.
* **AI Integration:** Recommender system suggesting relevant papers based on user history, and AI-powered assistant for question answering using uploaded documents.
* **Responsive & Modern UI:** Built with **React** and  **TailwindCSS** , optimized for desktop and mobile devices.
* **Backend & Database:** **Node.js** with **Express** and **SQL Server** for robust, secure data handling.
* **Secure & Scalable:** Password hashing, role-based access, and modular API architecture for maintainability.

## Technologies Used

* **Front-End:** React, TailwindCSS, React Router
* **Back-End:** Node.js, Express.js, JWT Authentication, bcrypt, CORS
* **Database:** SQL Server
* **AI & Recommender Integration:** Python-based recommender system, RAG/chatbot assistant
* **Other:** Environment management via dotenv, secure file handling, logging middleware

## Purpose & Goals

Insight is designed to **streamline academic research** by integrating advanced search, recommendations, and AI-driven insights within a single platform. Its goal is to improve research discovery, enhance collaboration, and provide actionable insights for users in a professional and scientifically structured environment.
